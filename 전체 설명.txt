** 페이지는 크게 4개 **


1. 메인 화면 : 간단하게 앱 기능 소개, 앱 사용 방법 등 공지
2. 앱 (개발) 정보 화면 : 앱 개발 과정과 더불어 앱의 기초 정보 설명
3. 고객 추천 모델 페이지 : 고객이 정보를 입력하면 그에 따른 차량 종류 추천하는 시스템
    - 조금 더 비싼 차량을 신용 카드로 구매할 시 10% 포인트로 제공한다는 것 공지하면서, 해당 차량 추천
    - 이때, 전기차가 아닌 차종 선택 시 대안이 될 수 있는 전기차 추천 동반 : 지역에 따른 보조금 차이 반영하여 추천
4. 기업(현차)의 데이터 분석 페이지 : 기업이 현재까지의 데이터 기반으로 마케팅 전략 구축
    - 차량 1회만 구매한 일반 회원에 대해 차량 재구매 시 할인 혜택 제공된다는 메일 발송 -> 실제 메일 발송은 X, 이게 가능하다는 것 보여주는 정도
    - 차량 (연료 구분)별로 어느 지역에서 많이 판매되었는지 확인 -> 해당 지역에서 전시장 및 지점 확장
        - 전반적으로 어떤 연료 타입의 차량이 많이 판매되었는지 확인하고 추후 차량 개발 전략 수정 가능
    - 구매 기준 시점 따른 연료 타입별 차량 구입 건수 확인 -> 어떤 차량의 판매가 추후 증가할 것인지 예측 가능
        - 수소차 판매량 감소 양상 : 2024년 4분기부터 지속된 수소 폭발 사고 파장
    - 고객 구분(개인, 법인)별 차량 판매 대수 확인
        - 2023년 2분기 및 2024년 3분기에 판매량 급감하는 양상
        - 비트코인 폭등으로 인해 제조업계 반도체 유통 급감 -> 반도체 수급 어려움 발생
    - 연령대별 차량 구매 횟수 분석 : 어떤 연령층에서 차량 구매 수요가 높은가
        - 20대 초반, 30대 중반, 50대 초반, 60대 후반에 증가하는 양상
        - 고객들의 차량의 교체 주기가 약 15년임을 확인 가능
            - 해당 연령대의 고객의 선호 차량 유형을 파악하여 타겟팅 가능
